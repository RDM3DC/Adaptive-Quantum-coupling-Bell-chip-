Bellchip Scripts
================

Included:
- bell_chip_three_qubit.py  -> 3-qubit A-B-C chain with local+shared memory; plots GHZ fidelity.
- bell_chip_two_qubit_ibm.py -> 2-qubit shared-memory simulator with IBM-style units + CLI (grid search).
- bell_chip_two_qubit.py     -> 2-qubit minimal example (if present).

Usage examples:
  python bell_chip_three_qubit.py
  python bell_chip_two_qubit_ibm.py --delta-ghz 5 --omega-ghz 0.1 --lam 0.15 --mu-ratio 0.1 --mode rotframe --grid

If you need a single-file copy, you can also copy-paste from chat, or grab files individually from this zip.
